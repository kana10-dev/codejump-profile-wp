## 🛠 習得・活用テクニック

### 1. WordPress開発環境の構築
`Local` をインストールし、ローカルサーバー上でWordPressを動作させる環境構築の手順を習得。

### 2. PHPによる動的サイトの実装
PHPの基礎を1から学習。静的なHTML/CSSをWordPressのテーマ構造（index.php, style.css等）へ分解し、動的にコンテンツを出力するテンプレートタグの使い方を習得した。

### 3. WordPressテーマの基本構造
WordPressテーマとして認識させるために必要なファイル構成を理解し、管理画面から操作可能なオリジナルテーマの土台を作成。

### 4. 静的サイトからの脱却
これまでの模写練習で培ったHTML/CSSのスキルをベースに、CMS（コンテンツ管理システム）を介した更新性の高いサイト制作のフローを実践。
